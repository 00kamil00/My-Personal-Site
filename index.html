My Projects

MemeBot (Python + Discord API)
A simple Discord bot that responds to user commands.
It can, for example, send greetings or random memes.
This was my first project using an API, which helped me understand the basics of interaction between applications.


Area Calculator (Python)
A console application for calculating the area of basic geometric shapes: triangle, rectangle, square, and circle.
The program takes user input (e.g., length, width) and returns the result.
This project helped me practice handling input and implementing simple mathematical algorithms in Python.


Pomodoro App (HTML/CSS/JS)
Pomodoro App is a simple and intuitive tool designed to support the Pomodoro technique for better time management.
The screen displays a 25-minute timer inside a circle, and when the Start button is pressed, the countdown begins.
Once the time runs out, a sound notification signals the end of the work session.
The app was built using HTML, CSS, and JavaScript and is part of my personal portfolio.
